# Custom Lightweight Web Browser (Python)

A custom-built web browser developed in Python as a lightweight, fully customizable alternative to mainstream browsers. The project was motivated by two common issues with browsers like Chrome: high RAM usage and limited user control over privacy and data handling.

This project also integrates a [motion-activated security camera system](https://github.com/KrishParmar2804/security-camera-system), combining everyday browsing with live camera monitoring in a single application.

## Features

- Lightweight core with minimal RAM footprint compared to mainstream browsers
- Fully customizable interface and behavior
- Integrated live view from the connected security camera system
- Navbar functionality with added access for Camera baked in.
- (x) to kill camera.

## Tech Stack

- Python
- GUI framework -PyQt5 with QtWebEngine
